Video Game Finder
Created By: Claudio Sciotto & John Torres

DataCollector: Java Code that connects to the IGDB API and retrieves a full data base of video games to a text file, storing the name, ID, genres and ratings

FilterData: Organizes the collected data from the previous programs into an easy to read cvs-like text files

Algorithms: Contains the entire data structures and searching algorithms, and requires input from filterData
